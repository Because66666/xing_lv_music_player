# Xing lv music player

星旅音乐播放器：一款现代星际主题的唱片音乐播放器，带有音频可视化功能，专为内容创作者录制音乐而设计。

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
